# 🛒 BlinkIt Sales Dashboard

An interactive sales analysis dashboard built using **Python, Pandas, and Matplotlib**.  
This project explores BlinkIt's product performance, outlet trends, and customer insights through visual analytics.

---

## 📌 Business Objective

The goal of this project is to:

- Analyze overall sales performance
- Compare product categories and fat content impact
- Identify top-performing item types
- Study outlet establishment trends
- Build a professional data dashboard using Python

## 📈 Dashboard Visualizations

### 🥧 Sales by Fat Content (Donut Chart)
- Shows contribution of Low Fat vs Regular products.
- Helps understand customer product preferences.

### 📊 Sales by Item Type (Horizontal Bar Chart)
- Highlights top-performing categories.
- Identifies low-performing segments.

### 📈 Sales by Establishment Year (Line Chart)
- Analyzes outlet age vs revenue performance.
- Reveals growth trends over time.

---

## 🧹 Data Cleaning Performed

- Standardized fat content values
- Handled missing values in Item Weight
- Verified dataset consistency

## 🛠 Technologies Used

## 🔎 Key Insights

- Low Fat products dominate overall sales.
- Certain item types significantly outperform others.
- Older outlets tend to generate higher revenue.
- Sales trends show stable performance growth.
